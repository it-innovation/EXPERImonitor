Packaging the ECC component (using Maven 3.x+)
----------------------------------------------

Build docs:

  cd doc
  make html

Obtain and install third party jar into maven:

  <get library>
  cd thirdPartyLibs
  installLibraries.sh/bat

Make source release (use -o option if too slow):

  mvn package assembly:assembly -Preleasesrc

Make binary release (for version 2.2):

  mvn package assembly:assembly -Preleasebin

this will create "target/experimedia-arch-ecc-2.2-bin". Change into:

  target/experimedia-arch-ecc-2.2-bin/bin

and run:

  zip -d EccService-2.2.war WEB-INF/lib/owlim-lite-5.4.jar

then create a zip file from experimedia-arch-ecc-2.2-bin folder manually:

  cd target
  zip -r experimedia-arch-ecc-2.2-bin.zip experimedia-arch-ecc-2.2-bin
